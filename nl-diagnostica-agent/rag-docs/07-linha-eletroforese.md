# Linha Eletroforese Capilar (V8 / SPIFE) — Produtos, finalidade e termos técnicos

**file_id:** LINHA-ELETROFORESE-01
**Finalidade:** detalhar a linha de Eletroforese para o agente justificar match item a item.
**Marca:** Helena Laboratories.

## O que é
A NL Diagnóstica fornece o **V8**, sistema de **eletroforese capilar automatizado de 8 canais**, para preparação, separação e análise de amostras humanas (**soro, urina e líquor**). O V8 também atua como **pipetador automático** para o sistema **SPIFE** (gel de agarose). Usa kits dedicados e oferece multi-ensaio, carregamento contínuo e reanálise inteligente de amostras anormais.

## Produtos / ensaios

### 1. Sistema de eletroforese capilar (V8)
- **Finalidade:** plataforma automatizada de eletroforese para os ensaios abaixo.
- **Termos em editais:** eletroforese, eletroforese capilar, eletroforese automatizada, gel de agarose (SPIFE), sistema de eletroforese.
- Normalmente licitado em **comodato** com fornecimento de kits/reagentes.

### 2. Proteínas séricas — V8 SPE (proteinograma)
- **Finalidade:** fracionamento de proteínas séricas (proteinograma).
- **Termos em editais:** proteínas séricas, proteinograma, fracionamento de proteínas, eletroforese de proteínas, SPE.

### 3. Hemoglobinas — V8 Hemoglobin IEF
- **Finalidade:** triagem de **hemoglobinopatias** por isoeletrofocalização (IEF).
- **Termos em editais:** eletroforese de hemoglobina, hemoglobinopatia, hemoglobina fetal, isoeletrofocalização, IEF.
- **Cuidado:** **não** confundir com **hemoglobina glicada (HbA1c)** — esta é bioquímica e **não** é desta linha. Só casar quando o contexto for eletroforese/hemoglobinopatia.

### 4. Imunofixação / Imunodeslocamento — V8 ID
- **Finalidade:** imunofixação/imunodeslocamento (IgG, IgA, IgM, Kappa e Lambda); investigação de **gamopatias monoclonais**.
- **Termos em editais:** imunofixação, imunodeslocamento, imunossubtração, proteína/componente monoclonal, gamopatia, cadeias leves kappa/lambda.

## Como o agente deve usar isto
1. Se o item cita **eletroforese, proteinograma, hemoglobinopatia, imunofixação** → linha Eletroforese (fornecível).
2. Se o item é **soro, urina ou líquor** mas o método não é eletroforese (ex.: dosagem bioquímica, urinálise comum) → avaliar como **fora de escopo**, salvo se houver outro produto no catálogo.
3. Se for "sistema de eletroforese em comodato com kits" → trata-se do conjunto V8 + reagentes.
