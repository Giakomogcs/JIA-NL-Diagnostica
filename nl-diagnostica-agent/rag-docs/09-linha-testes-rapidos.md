# Linha Testes Rápidos — Produtos, finalidade e termos técnicos

**file_id:** LINHA-TESTES-RAPIDOS-01
**Finalidade:** detalhar a linha de Testes Rápidos para o agente justificar match item a item.

## O que é

Testes **imunocromatográficos** (sorologia rápida) de leitura visual. O item de catálogo atual é o **Cellex qSARS-CoV-2 IgG/IgM** (COVID-19), aprovado por FDA e ANVISA.

## Produto

### Teste rápido COVID-19 — Cellex qSARS-CoV-2 IgG/IgM

- **Finalidade:** detecção qualitativa de anticorpos **IgG/IgM anti-SARS-CoV-2**.
- **Termos em editais:** teste rápido COVID-19, SARS-CoV-2, sorologia COVID, anticorpo IgG/IgM, imunocromatográfico para coronavírus.

## Como o agente deve usar isto

1. Só casar quando o item pedir especificamente **teste rápido de COVID-19 / SARS-CoV-2** (linha estreita).
2. **Atenção:** "teste rápido" é um termo genérico — testes rápidos de **gravidez, glicemia, dengue, HIV, sífilis** etc. **não** estão no catálogo. **Não** assuma que a empresa fornece qualquer teste rápido; só os que estiverem cadastrados.
3. Se aparecer demanda recorrente por outro teste rápido, sinalize para **cadastrar no catálogo** ou **consultar o comercial**.
